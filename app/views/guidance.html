1. What a SIC code is

A SIC code is a 5-digit code which provides Companies House with a description of your company’s nature of business. It allows us to track the number of businesses that are active in different industries.

When you register a new company, you’ll need to provide at least one SIC code to describe what it does.

We use a reduced version of the full list of codes (opens in a new tab) available from the Office of National Statistics.

2. How to choose a SIC code

You should use the SIC code that best describes the goods or services your company produces or sells. It does not have to be an exact match. You can use up to four SIC codes if your company provides different goods or services.

3. How to change a SIC code

You can only change a SIC code when you file a confirmation statement. You can file this early if you need to change your code before the confirmation statement is due.
